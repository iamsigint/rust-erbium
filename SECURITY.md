## ğŸ† Bug Bounty Program

We run a bug bounty program to encourage responsible disclosure of security vulnerabilities.

### Rewards

- **Critical**: $5,000 - $10,000
- **High**: $2,000 - $5,000
- **Medium**: $500 - $2,000
- **Low**: $100 - $500

### Scope

The bug bounty program covers:
- Erbium core blockchain implementation
- Network protocol and P2P communication
- Smart contract execution engine
- Cross-chain bridge implementations
- API endpoints and interfaces

### Out of Scope

- Third-party dependencies (report to respective projects)
- Known issues already documented
- Social engineering attacks
- Physical security issues
- DDoS attacks without technical vulnerability
